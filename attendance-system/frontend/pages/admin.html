How this works

Quick guide

  1. Create a class and assign a teacher.
  2. Add students to the class — each gets a unique QR code automatically.
  3. Print or share each student's QR (their digital/physical ID card).
  4. Open the Scan Station during class — students scan in and are marked present instantly.
  5. Check Reports for attendance percentage and defaulters (below 75%).
